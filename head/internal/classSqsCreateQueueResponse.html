bool SqsCreateQueueResponse::isValid ( ) const
virtual

Is this AWS response valid?

This base implementation simply checks for the absense of network and XML parsing errors. Derived classes may override this function to check other error types, and/or apply additional validation rules.

Returns
true if this response is considered valid, false otherwise.
Note
This function may be equivalent to !hasError, however, it is not safe for this function's implementation to be just that, since derived classes may assume the inverse, resulting in an infinite cycle between the two functions.

void SqsCreateQueueResponse::parseSuccess ( QIODevice &  response)
protectedvirtualslot

Parse an SQS CreateQueue response.

This implementation parses SQS CreateQueueResponse elements, as defined by http://queue.amazonaws.com/doc/2012-11-05/QueueService.wsdl

For example:

1 <CreateQueueResponse>
2  <CreateQueueResult>
3  <QueueUrl>http://sqs.us-east-1.amazonaws.com/123456789012/testQueue</QueueUrl>
4  </CreateQueueResult>
5  <ResponseMetadata>
6  <RequestId>7a62c49f-347e-4fc4-9331-6e8e7a96aa73</RequestId>
7  </ResponseMetadata>
8 </CreateQueueResponse>
