Detailed Description

Represents a request to be sent (via a client class) to an Amazon service.

Typically, a user would pass a derived class (such as SqsCreateQueueRequest) to an AwsAbstractClient-derived class (such as SqsClient) to send a request to an AWS service.

QNetworkRequest AwsAbstractRequest::networkRequest ( const QUrl &  endpoint,
const AwsAbstractSignature &  signature,
const AwsAbstractCredentials &  credentials 
) const
virtual

Construct a QNetworkRequest for this AWS request.

This base implementation used the protected unsignedRequest function to fetch the basic request to be sent, then uses signature and credentials to sign the request before returning to the caller.

Derived classes may override this if their specific AWS service required more advanced logic (or disallow signing, for example).

Parameters
endpointAWS endpoint the request is to be for.
signatureAWS signature class for signing the request.
credentialsAWS credentials to use for signing the request.
Returns
A valid, signed (where appropriate) request that can be sent to via a network access manager to perform the configured request.
